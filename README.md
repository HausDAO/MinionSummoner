# Minions

These minions support the NFT DAOHaus boost.

```
yarn install
yarn test

```

## NFT Tribute Minion

Applicant proposes NFT for shares, or NFT for treasury funds.

If proposal passes, NFT moves to Vanilla minion. If fails or cancelled, sends back to applicant.
